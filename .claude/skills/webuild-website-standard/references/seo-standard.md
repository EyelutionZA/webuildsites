# SEO and Conversion Standard

## Baseline SEO

Every production website should have:

- one clear H1 per important page
- semantic HTML
- page title
- meta description
- Open Graph metadata where relevant
- sitemap
- robots
- descriptive image alt text
- canonical URLs where relevant
- local business schema where relevant

## Conversion Baseline

Every important page should have:

- clear audience
- clear problem/value proposition
- primary CTA
- secondary CTA where useful
- trust proof
- contact path
- no dead-end sections

## Local Business Notes

For local South African service businesses, check:

- city/service area mentions where natural
- phone/WhatsApp visibility
- contact form
- Google Business Profile link if available
- reviews/testimonials
- service-specific pages where valuable

## Red Flags

- multiple H1s without reason
- no meta descriptions
- image-only headings
- CTAs hidden below the fold
- no trust proof
- no form/phone/WhatsApp path
- generic AI copy with no local relevance
