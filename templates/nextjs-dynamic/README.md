# __WEBUILD_PROJECT_TITLE__

A dynamic Webuild website generated from the `nextjs-dynamic` template.

## Commands

```bash
npm install
npm run dev
npm run build
npm run start
npm run webuild:detect
npm run webuild:validate
npm run webuild:report
```

## Hosting Mode

`dynamic`

This project includes API route form handling and a health endpoint. It needs a runtime such as Vercel Functions, Node, or Coolify-compatible hosting.

## Required Environment Variables

```env
FORM_WEBHOOK_URL=
```
